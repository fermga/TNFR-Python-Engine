**Status**: Production-ready ✅

## Overview

This guide documents the **complete self-optimization integration** across TNFR modules, providing a unified workflow for:

1. **Paley Factorization** (original integration)
2. **Pattern Discovery** (mathematical pattern engine)
3. **Fractal Partitioning** (coherence-based network partitioning)
4. **Batch Certificate Processing** (analysis workflows)

All integrations use the **centralized self-optimization support** module (`tnfr_factorization.self_opt_support`) and expose clean APIs via the **TNFR SDK** (`tnfr.sdk`).

## Usage Examples

### 1. Pattern Discovery Optimization

```python
from tnfr.engines.pattern_discovery import TNFREmergentPatternEngine
from tnfr.sdk import run_pattern_discovery_optimization
import networkx as nx

# Create and analyze network
G = nx.karate_club_graph()
for node in G.nodes():
    G.nodes[node]["vf"] = 1.0
    G.nodes[node]["phase"] = 0.0
    G.nodes[node]["EPI"] = 1.0

# Discover patterns
engine = TNFREmergentPatternEngine()
discovery_result = engine.discover_all_patterns(G)

# Export manifest
manifest_info = engine.export_pattern_manifest(
    G=G,
    discovery_result=discovery_result,
    output_dir="results/patterns",
    partition_id="experiment_42",
)

# Run self-optimization
opt_result = run_pattern_discovery_optimization(
    manifest_path=manifest_info["manifest_absolute"],
    manifest_summary_path=manifest_info["summary_absolute"],
    base_name="pattern_opt_42",
)

print(f"Optimization result: {opt_result}")
```

### 2. Fractal Partition Optimization

```python
from tnfr.parallel import FractalPartitioner
from tnfr.sdk import run_fractal_partition_optimization
import networkx as nx

# Create network
G = nx.karate_club_graph()
for node in G.nodes():
    G.nodes[node]["vf"] = 1.0
    G.nodes[node]["phase"] = 0.0

# Partition with manifest
partitioner = FractalPartitioner(max_partition_size=10)
result = partitioner.partition_with_manifest(
    graph=G,
    output_dir="results/partitions",
    partition_id="network_42",
)

# Run self-optimization
opt_result = run_fractal_partition_optimization(
    manifest_path=result["manifest_absolute"],
    manifest_summary_path=result["summary_absolute"],
    base_name="partition_opt_42",
)

print(f"Partitions: {len(result['partitions'])}")
print(f"Optimization result: {opt_result}")
```

### 3. Batch Certificate Optimization

```python
from factorization_lab.analysis.analyze_patterns import generate_optimization_manifest
from tnfr.sdk import run_batch_certificate_optimization

# Generate optimization manifest from certificate analysis
manifest_info = generate_optimization_manifest(
    certificate_manifest_path="results/certificates/certificate_manifest.json",
    output_dir="results/batch_optimization",
    batch_id="batch_42",
    certificate_filter={"coherence_ratio_max": {"min": 0.7}},
)

# Run self-optimization on batch
opt_result = run_batch_certificate_optimization(
    manifest_path=manifest_info["manifest_absolute"],
    manifest_summary_path=manifest_info["summary_absolute"],
    base_name="batch_opt_42",
)

print(f"Optimization result: {opt_result}")
```

## Implementation Checklist

When adding self-optimization to a new module:

- [ ] **Create manifest export method** that generates JSON with:
  - `operation_type` field (unique identifier)
  - `partition_id` or equivalent identifier
  - `telemetry` dict with coherence, sense_index
  - `network_metadata` with node/edge counts
  - Module-specific data fields

- [ ] **Write summary file** alongside manifest with key metrics

- [ ] **Add SDK wrapper function** in `src/tnfr/sdk/self_opt.py`:
  ```python
  def run_<module>_optimization(...):
      return run_partition_self_optimization(
          ...,
          operation_type="<module>_operation"
      )
  ```

- [ ] **Expose in SDK** via `src/tnfr/sdk/__init__.py`:
  - Add to `__all__`
  - Add lazy import in `__getattr__`

- [ ] **Create unit tests** in appropriate `tests/` subdirectory:
  - Test manifest file creation
  - Test manifest structure validation
  - Test SDK wrapper integration
  - Test telemetry computation

- [ ] **Update documentation** in this file with:
  - Manifest format specification
  - Usage examples
  - Test commands

## Troubleshooting

### Issue: "Self-optimization helpers not available"

**Cause**: `tnfr_factorization` package not installed or not in Python path.

**Solution**:
```bash
# Ensure factorization-lab is in PYTHONPATH
export PYTHONPATH="${PYTHONPATH}:$(pwd)/factorization-lab"

# Or install in development mode
pip install -e factorization-lab/
```

### Issue: "Manifest validation failed"

**Cause**: Missing required fields in manifest JSON.

**Solution**: Ensure manifest includes all common fields:
- `operation_type`
- `partition_id` (or equivalent)
- `telemetry` dict
- `network_metadata` dict

### Issue: "CLI runners not found"

**Cause**: Scripts not available or incorrect import paths.

**Solution**: Verify scripts exist:
```bash
ls scripts/run_self_optimization.py
ls scripts/run_self_opt_validation.py
```

## Migration Guide

### From Direct CLI Invocation

**Before**:
```python
import subprocess
subprocess.run([
    "python", "scripts/run_self_optimization.py",
    "--manifest", "manifest.json",
    "--output-dir", "results/"
])
```

**After**:
```python
from tnfr.sdk import run_pattern_discovery_optimization

result = run_pattern_discovery_optimization(
    manifest_path="manifest.json",
    base_name="experiment_42"
)
```

### From Custom Self-Optimization Logic

**Before**:
```python
# Custom pipeline in module
def my_custom_self_opt(G, output_dir):
    # 50 lines of pipeline logic...
```

**After**:
```python
# Export manifest
manifest_info = engine.export_pattern_manifest(
    G=G, discovery_result=result,
    output_dir=output_dir, partition_id="exp_42"
)

# Use centralized pipeline
opt_result = run_pattern_discovery_optimization(
    manifest_path=manifest_info["manifest_absolute"]
)
```
