Step 2: Core Concepts

What is TNFR?

TNFR (Resonant Fractal Nature Theory) models reality as resonant patterns rather than isolated objects. Think of a choir: each voice maintains its unique identity while synchronizing with others to create harmony.

Key Elements

Resonant Fractal Node (NFR)

The minimum unit of structural coherence. Each node has:

  • EPI: Primary Information Structure (the coherent "shape")
  • ŒΩf: Structural frequency in Hz_str (reorganization rate)
  • Phase œÜ: Relative synchrony with the network

The Fundamental Equation

‚àÇEPI / ‚àÇt = ŒΩf ¬∑ ŒîNFR(t)

This equation captures how structures change: the rate of change (‚àÇEPI/‚àÇt) is proportional to both the structural frequency (ŒΩf) and the reorganization gradient (ŒîNFR).

Structural Operators (13 Canonical)

The only way to modify nodes in TNFR:

  • Emission (AL): Initiates a resonant pattern
  • Reception (EN): Receives and integrates external patterns
  • Coherence (IL): Stabilizes structural form
  • Dissonance (OZ): Introduces controlled instability
  • Coupling (UM): Creates structural links between nodes
  • Resonance (RA): Amplifies and propagates patterns
  • Silence (SHA): Temporarily freezes evolution
  • Expansion (VAL): Increases structural complexity
  • Contraction (NUL): Reduces structural complexity
  • Self-organization (THOL): Spontaneous pattern formation
  • Mutation (ZHIR): Phase transformation
  • Transition (NAV): Movement between structural states
  • Recursivity (REMESH): Nested operator application

Step 3: Create Your First TNFR Network

Let's create a simple network in just 3 lines of code:

from tnfr.sdk import TNFRNetwork

# 1. Create a network
network = TNFRNetwork("hello_world")

# 2. Add nodes, connect them, apply operators, measure
results = network.add_nodes(10) \
                 .connect_nodes(0.3, "random") \
                 .apply_sequence("basic_activation", repeat=3) \
                 .measure()

# 3. Display results
print(results.summary())

What just happened?

  • add_nodes(10): Creates 10 resonant nodes
  • connect_nodes(0.3, "random"): Randomly connects nodes with 30% probability
  • apply_sequence("basic_activation", repeat=3): Applies structural operators 3 times
  • measure(): Calculates coherence metrics (C(t), Si, ŒΩf)

Step 4: Understanding Health Metrics

Grammar 2.0 introduces quantitative health metrics to assess sequence quality. All metrics range from 0.0 (poor) to 1.0 (excellent).

Overall Health

Composite score combining all dimensions

0.8-1.0 Excellent
0.6-0.8 Good
0.4-0.6 Fair
<0.4 Poor

Coherence Index

Global sequential flow quality

  • Valid transitions between operators
  • Pattern structure recognition
  • Structural closure (proper ending)

Balance Score

Equilibrium between stabilizers and destabilizers

  • Stabilizers: coherence, silence, resonance
  • Destabilizers: dissonance
  • Ratio determines balance

Sustainability Index

Long-term viability of the pattern

  • Proper termination
  • Force equilibrium
  • Structural stability
